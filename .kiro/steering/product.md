# StreamAPI Product Overview

StreamAPI is a movie and TV show streaming platform that provides users with access to a comprehensive database of entertainment content. The application serves as a streaming API service that fetches movie and TV show data from TMDB (The Movie Database) and VidSrc.

## Core Features

- **Content Discovery**: Browse popular movies, TV shows, latest releases, and top-rated content
- **Search & Filter**: Advanced search functionality for finding specific movies and TV shows
- **User Management**: Complete authentication system with registration, login, password reset
- **Watchlist**: Personal watchlist management for saving favorite content
- **Watch History**: Track viewing progress and history across devices
- **Real-time Features**: WebSocket-powered live user count and watch party functionality
- **Torrent Integration**: Built-in torrent search and download capabilities
- **Newsletter**: Email subscription system for updates and notifications
- **Responsive Design**: Mobile-first design with Tailwind CSS

## Target Audience

Entertainment enthusiasts looking for a comprehensive streaming platform with social features and content discovery tools.

## Key APIs & Data Sources

- **TMDB API**: Primary source for movie and TV show metadata, images, and ratings
- **VidSrc**: Streaming source integration for video content
- **MongoDB**: User data, watch history, and application state storage
