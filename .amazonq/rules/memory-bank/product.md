# StreamAPI - Product Overview

## Project Purpose
StreamAPI is a comprehensive streaming platform that provides users with access to movies and TV shows through a modern web interface. The application serves as a bridge between The Movie Database (TMDB) API and various streaming sources, offering a Netflix-like experience for browsing and watching content.

## Core Value Proposition
- **Unified Content Discovery**: Aggregates movie and TV show data from TMDB with streaming capabilities
- **Modern User Experience**: Responsive, Tailwind CSS-powered interface with intuitive navigation
- **Multi-Platform Deployment**: Supports deployment on Vercel, Render, Docker, and local development
- **User Management**: Complete authentication system with profiles, watchlists, and viewing history
- **Real-time Features**: WebSocket-powered watch parties and live user tracking

## Key Features & Capabilities

### Content Browsing
- Browse popular, latest, and top-rated movies and TV shows
- Advanced search functionality for both movies and TV content
- Trending content discovery with weekly/daily filters
- Similar content recommendations
- Detailed movie/TV show information with OMDB integration

### User Features
- User registration and authentication system
- Personal watchlist management
- Watch history tracking with progress saving
- User profiles with customizable avatars
- Achievement system with badges and points

### Streaming & Playback
- Integrated video player with VidSrc embedding
- Multiple streaming source support (VidSrc, GoDrive)
- Episode-by-episode TV show viewing
- Continue watching functionality
- Watch party system for synchronized viewing

### Social Features
- Real-time watch parties with chat functionality
- Active user count tracking
- Newsletter subscription system
- Contact form with email notifications

### Technical Features
- RESTful API endpoints for content and user management
- WebSocket integration for real-time features
- Email system with templated notifications
- File upload support for profile pictures
- Torrent integration for alternative content access

## Target Users
- **Movie & TV Enthusiasts**: Users seeking a centralized platform for content discovery and streaming
- **Social Viewers**: Groups wanting to watch content together through watch parties
- **Content Curators**: Users who maintain watchlists and track viewing progress
- **Developers**: Those interested in streaming platform architecture and TMDB API integration

## Use Cases
- Personal entertainment streaming and content discovery
- Social movie nights with synchronized playback
- Content research and information gathering
- Educational purposes for understanding streaming platform development
- API integration examples for movie/TV data handling
